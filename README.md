# Macio Wearhouse

A service where you can sell, or buy a car.

## Features

-   User login/signup
-   Searching for offers
-   Filtering & Sorting offers
-   Creating offers
-   Bookmarking offers

## Tech Stack

**Client:** Angular, TailwindCSS

**Server:** Firebase
